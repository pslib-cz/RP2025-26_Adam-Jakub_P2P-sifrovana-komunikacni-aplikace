# 🚀 Peer4Mans

Moderní P2P komunikační aplikace zaměřená na soukromí, decentralizaci a přímé spojení mezi uživateli.

## 🧠 O projektu

Peer4Mans je aplikace umožňující:

- přímou komunikaci mezi uživateli (P2P)
- minimalizaci závislosti na serverech
- ukládání základních dat o uživateli a jeho kontaktech

Cílem je vytvořit bezpečnější alternativu ke klasickým chatovacím aplikacím.

---

## ⚙️ Funkce

- 🔗 Přímé spojení user ↔ user
- 👥 Seznam kontaktů (aktivní / neaktivní)
- 💬 Messaging (online i offline doručení)
- 🧾 Historie konverzací
- 🔐 Důraz na soukromí

---

## 🧱 Tech Stack

- **Frontend:** React + TypeScript
- **Backend:** Node.js (WebSocket)
- **Databáze:** SQLite

---

## 🖥️ UX / UI koncept

- minimalistický design
- real-time indikace aktivity uživatelů
- jednoduchý dashboard pro správu kontaktů
- důraz na rychlost a přehlednost

---

## 📡 Architektura

- P2P komunikace mezi klienty
- server pouze jako signalizační vrstva a autentizace
- ukládání metadat lokálně

---

## 🔐 Bezpečnost

- žádné centrální ukládání zpráv
- možnost šifrování komunikace
- ochrana identity uživatele
